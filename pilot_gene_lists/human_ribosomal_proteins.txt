# Expected CodonScope results:
#   Mode 1: Strong codon bias; enrichment for C-ending codons (GC3 bias)

#   Mode 5: Mixed signal — both AA-driven and synonymous-driven components.

#   Mode 6: Low RSCU correlation with yeast orthologs (~0.1-0.15) despite both
